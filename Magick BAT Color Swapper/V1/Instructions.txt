Image Magick Must be installed for this script to function!
You can download it here: https://imagemagick.org/download/

Be careful with this script! It won't destroy your computer, but if you choose the wrong folder path you could accidentally mess up images you didn't want it to touch!

This is a Windows Batch Script to change specific colors on images in multiple subfolders using ImageMagick.
This script will recursively go through ALL subfolders inside the folder path you enter.
If you are confident enough to modify this file, go right ahead! Just don't blame me if something goes wrong.

Don't enter the path directly to one image, set it to a folder. Again, it will do every subfolder inside the folder you choose.
Path can be in quotation marks or not. Extention can have dot or not. Colors are in Hex Format. Examples Below.

"C:\Users\Example\Downloads\" will modify all images contained in the downloads folder and subfolders.
.png and "png" are both acceptable ways to enter the Extension.

"e52b50" and #702963 are both acceptable ways to enter the color values. You can just spam enter once you add all the colors you want to change.
Make sure to not add an old color without choosing a new one! Leaving an empty value for an old or new color will choose FFFFFF(White) instead.

The magick.exe path should look something like this! "C:\Program Files\ImageMagick-7.1.2-Q16-HDRI\magick.exe"
